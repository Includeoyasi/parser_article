1) Для успешного запуска программы необходимо установить python 2.7 в корень диска C
	скачать программу можно по ссылке: https://www.python.org/downloads/
2) После установки python запустите install.bat, чтобы установить необходимые библиотеки(requests и BeautifulSoup).
3) Если есть необходимость, то можете изменить файл настройки settings.py
3) Запустите run.bat и введите полный адрес статьи
	
В рузельтате успешного выполнения программы в папке articles появится файл с текстом статьи.
